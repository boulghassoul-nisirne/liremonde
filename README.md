LireMonde 📚

Application web interactive de gestion et découverte de livres développée en HTML, CSS, JavaScript avec une API REST simulée via JSON Server.

🎯 Objectif du projet

LireMonde permet aux utilisateurs de :

Consulter une liste de livres
Rechercher des livres par titre ou auteur
Filtrer les livres par genre
Voir les détails d’un livre
Ajouter des livres à une liste “À lire”
Gérer les livres depuis un espace administrateur
🛠 Technologies utilisées
HTML5
CSS3
JavaScript Vanilla
JSON Server
API REST simulée

⚙️ Installation du projet
1️⃣ Cloner le projet
git clone https://github.com/votre-username/liremonde.git
2️⃣ Accéder au dossier
cd liremonde
3️⃣ Installer JSON Server
npm install
4️⃣ Lancer le serveur API
npm start

Le serveur démarre sur :

http://localhost:3000/books
